# 🏏 IPL Data Analysis Dashboard

An interactive web application built with Streamlit for analyzing and visualizing historical Indian Premier League (IPL) match data. This dashboard processes raw JSON match data, displays key statistics, and provides overall, season-wise, and team-wise insights.

*(You can replace this with a screenshot of your running application)*

-----

## \#\# Features

  * **Interactive Visualizations**: Clean and modern charts and tables powered by Plotly.
  * **Comprehensive Analysis**: Explore overall IPL statistics, or dive deep with season-wise and team-wise filters.
  * **Automatic Data Processing**: On startup, the app automatically processes raw JSON data into clean CSV files.
  * **Dynamic UI**: The interface is built to be efficient, avoiding repetitive code and loading data dynamically.
  * **Containerized**: Includes a `Dockerfile` for easy and consistent deployment on any machine.

-----

## \#\# 📁 Project Structure

The repository is organized to separate source data, processed data, assets, and application code.

```
.
├── app.py                 # The main Streamlit application script
├── helper.py              # Helper functions for data analysis and plotting
├── requirements.txt       # Python dependencies
├── Dockerfile             # Instructions to build the Docker image
├── .dockerignore          # Files to ignore in the Docker build
├── json/                  # Source data
│   ├── ipl_match/         # Raw JSON file for each match
│   └── ipl_team_info/     # JSON file with team metadata
├── csv/                   # Processed data (auto-generated by the app)

│   └── teams_info.csv
├── images/                # App images and team logos
│   ├── 1.png
│   └── csk.jpg
└── audio/                 # Audio files
    └── IPL-theme-RMX.wav
```

## \#\# 🔧 Setup and Installation

Follow these steps to set up and run the project locally.

### \#\#\# Prerequisites

  * Python 3.8+
  * pip package manager

### \#\#\# Installation

1.  **Clone the repository:**

    ```sh
    git clone <your-repository-url>
    cd IPL-Data-Analysis-master
    ```

2.  **Create and activate a virtual environment (recommended):**

    ```sh
    python3 -m venv venv
    source venv/bin/activate
    # On Windows, use: venv\Scripts\activate
    ```

3.  **Install the required dependencies:**

    ```sh
    pip install -r requirements.txt
    ```

## \#\# 🚀 Usage

You can run the application either locally using Streamlit or as a container using Docker.

### \#\#\# 1. Running Locally

Once the setup is complete, run the following command from the project's root directory:

```sh
streamlit run app.py
```

Open your web browser and go to **`http://localhost:8501`**.

### \#\#\# 2. Running with Docker

This is the easiest way to run the application without worrying about local dependencies.

1.  **Build the Docker image:**

    ```sh
    docker build -t ipl-dashboard .
    ```

2.  **Run the Docker container:**

    ```sh
    docker run -p 8501:8501 ipl-dashboard
    ```

Open your web browser and go to **`http://localhost:8501`**.
